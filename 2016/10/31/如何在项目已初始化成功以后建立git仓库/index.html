如何在项目已初始化成功以后建立git仓库

  1. 首页要在github上创建好仓库,并且把git地址给拷贝一些,我就拿我的要上传的项目做例子吧,ssh的地址是:git@github.com:vshiron/html-css-js-dome.git
  2. 然后你就需要在本地创建一个文件,把你需要的东西先考进去吧,然后打开终端,cd进入你的目录:cd /Users/vshiron/Desktop/html学习/jsdome
  3. 在该目录下使用git的初始化命令:git init
  4. 以后的每一次增加目录里面的内容,都必须要用该命令来添加到本地仓库,用”.”是代表通配符,命令如下:git add .,成功后不会显示任何东西
  5. 然后就是提交代码的信息,修改了什么就顺便写上去,第一次初始化就可以写first啦:git commit -m "first commit"
    大概会显示下面内容

  1. 完事了以后,就要添加一个远程本地的仓库:git remote add origin git@github.com:vshiron/html-css-js-dome.git,完事后没有任何代码显示

  2. 接下来再输入另外一个代码,就是我们希望把代码推送到服务器上的命令:git push origin master
    这个时候会显示如下信息,他大概的意思是告诉你,推送代码时要养成拉代码合并的习惯~

  1. 那么,我们就先拉一下代码吧,运行命令:git pull origin master,拉完代码以后会显示下面信息:
  1. 操作完成上面那一步的话,我们可以继续使用该命令:git push origin master,这个时候可能会让你输入相应的账号和密码,输入就是了~

这个时候就已经完成项目的推送了~
